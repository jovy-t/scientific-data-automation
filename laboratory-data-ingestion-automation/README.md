# Laboratory Data Ingestion Automation

## Overview

External testing providers often deliver analytical results in inconsistent archive formats and file structures.

Common challenges include:

* Different archive formats
* Nested folder structures
* Multiple file types per sample
* Metadata stored in PDF reports
* Manual file organization

This project automates extraction, matching, and organization of incoming laboratory data packages.

## Workflow

Archive Package
→ Extraction
→ Folder Discovery
→ Metadata Extraction
→ File Matching
→ Sample Organization
→ Structured Output

## Features

* Automatic archive extraction
* Nested folder handling
* PDF metadata extraction
* Cross-file matching
* Automatic directory creation
* Batch processing
* Error handling
* Repeatable file organization
